# 🎬 Telegram Movie Forwarder Bot

A powerful Telegram bot that automatically indexes and forwards movies from channels with smart search, auto-correction, and admin panel.

## ✨ Features

- 🤖 Auto Indexing from multiple channels
- 🔍 Smart Search with auto-correction
- 📝 Similar Movies Suggestions
- 👑 Admin Panel with Buttons
- 💾 SQLite Database (Fast & Lightweight)
- ⌨️ Typing Indicators (Realistic feel)
- 🔄 Webhook Retry System
- 📊 Trending Movies Stats
- 🎯 Duplicate Detection
- 🔒 Environment Variables (No hardcoding)

## 🚀 Commands

| Command | Description |
|---------|-------------|
| `/start` | Welcome message |
| `/help` | Help menu |
| `/channels` | List all channels |
| `/admin` | Admin panel |
| `/stats` | Bot statistics |
| `/indexstats` | Indexing statistics |

## 📦 Installation

1. Clone repository
2. Copy `.env.example` to `.env` and fill credentials
3. Run `composer install`
4. Setup webhook: `https://your-domain.com/?setwebhook=1`
